\activity{\github{browningluke-archive/htn21-cactus}{Hack the North - CactUs}}{Vancouver, Canada}{Sep 2021}{
  \item	Prototyped a mental health app React Web App, using Google's speech-to-text and sentiment analysis APIs.
  \item Users can 'talk' to the cactus; where positive phrases would lead to growth, and negative to withering.
  \item Users' cactus growth is stored in a Firebase firestore, authenticated using Discord's OAuth2.
}
